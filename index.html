Hi there. My name is Eigen, let's talk

Eigen is Machine Learning @ Berkeley's entry into the inaugural Alexa Prize Competition. The goal of the competition is to create a social bot that can maintain a coherent conversation with a human for 20 minutes. We're leveraging the latest results in NLP, machine learning, and deep learning to make this a reality. We're currently working on a demo so you can try it out for yourself.
